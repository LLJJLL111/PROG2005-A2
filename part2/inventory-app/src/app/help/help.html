Help & Frequently Asked Questions

Basic Operation

Add new products via the Inventory page by filling in Item ID, Name, Category, Quantity, Price, and Supplier fields. Mark items as "Popular" for quick identification, then click "Add New Item" to save to your inventory list.

Stock Status Rules

Stock status is auto-calculated based on quantity: 0 units = Out of Stock, 1–4 units = Low Stock, 5+ units = In Stock. This helps you quickly identify items needing restocking.

Search Function

Use the Search page to filter inventory items by product name, ID, or category. Enter a keyword in the search box and click "Search Inventory" to view matching results (partial keyword matching is supported).

Data Management

  • All inventory data is stored locally on your device (see Privacy page for details)
  • Edit/delete functionality is coming in Version 2.0
  • Clear browser local storage to reset all inventory data

Common Issues

  • Form submission fails: Ensure all required fields are filled and values are positive
  • Inventory list is empty: Local storage may have been cleared (re-add products)
  • Navigation links not working: Restart the Angular server and clear browser cache

Note: The application works offline with no internet connection required. For further assistance, contact support at support@inventoryms.com.
